## Project Background
- 최근 재활병원 및 케어시설에서는 환자의 **낙상, 비정상적인 행동, 응급 상황**을 실시간으로 감지하고 대응할 수 있는 스마트 모니터링 시스템의 필요성이 급격히 증가하고 있다.  
- 기존 CCTV 기반 감시 방식은 단순한 영상 저장에 그쳐 **관리자의 즉각적인 상황 파악과 대응이 어렵다**는 한계를 가진다.  
- 본 연구에서는 **인공지능(AI)** 기반의 행동 분석과 **WebRTC 실시간 스트리밍**, **GraphQL 데이터 관리**, **MongoDB 로그 저장**을 결합하여 환자의 움직임을 실시간으로 감지하고, 위험 상황 시 관리자에게 즉시 경고를 제공하는 **지능형 모니터링 시스템**을 개발하였다.

---

## System Model

시스템은 다음 세 가지 주요 구성요소로 이루어진다.

1. **엣지 카메라 모듈**
   - YOLOv8 기반 객체 탐지 모델을 적용하여 낙상·움직임 패턴 분석  
   - 비정상적인 행동 감지 시 백엔드 서버로 이벤트 전송  

2. **백엔드 서버 (FastAPI + Node.js)**
   - WebSocket 및 WebRTC를 이용한 실시간 영상 스트리밍 구현  
   - MongoDB와 GraphQL을 통한 데이터 수집, 질의 및 관리  
   - 이벤트 로그 저장 및 관리자 대시보드로 전달  

3. **프론트엔드 대시보드 (React 기반)**
   - 실시간 영상 및 경고 알림 시각화  
   - 환자별 행동 통계와 이벤트 로그 조회 기능 제공  
   - 관리자 중심의 직관적인 UI/UX 설계

## Numerical Results
- 프로토타입 테스트를 통해 다음과 같은 성능 결과를 도출하였다.  
  - **실시간 영상 스트리밍 지연 시간:** 평균 **0.8초 이하**  
  - **낙상 및 비정상 행동 감지 정확도:** **90% 이상**  
  - **GraphQL 기반 데이터 처리 효율성:** 기존 REST API 대비 **약 20% 향상**  
  - **대시보드 반응 속도:** 이벤트 발생 후 **3초 이내 반영**

테스트는 로컬 네트워크 환경에서 진행되었으며,  
실제 병원 및 재활시설 환경에서도 적용 가능한 수준의 성능을 확인하였다.

---

## Conclusion
본 프로젝트를 통해 개발된 **AI 기반 실시간 환자 모니터링 시스템**은 기존 CCTV 시스템의 한계를 극복하고,  
낙상 및 이상 행동을 빠르게 탐지하여 **즉각적인 대응이 가능한 구조**를 구현하였다.

- 관리자에게 실시간 알림을 제공함으로써 환자 안전성을 향상시켰으며,  
- WebRTC와 GraphQL을 기반으로 안정적인 스트리밍 및 데이터 처리 구조를 검증하였다.  
- 향후에는 **모듈 경량화**와 **임베디드 환경 최적화**를 통해 병원, 가정, 재활센터 등 다양한 환경에서의 실사용을 목표로 하고 있다.
